Microblog
Microblog is a Flask-based web application that allows users to register, create posts, follow other users, and search content. It includes authentication, password reset via email, internationalization support, and full database migrations.

Features
User registration and authentication
Password reset via email
Create, edit, and delete posts
Follow and unfollow users
User profile pages
Full-text search
Internationalization (Ukrainian translation included)
Database migrations with Alembic
Logging support
Docker support

Tech Stack
Python 3
Flask
Flask-Login
Flask-WTF
Flask-Mail
Flask-Migrate (Alembic)
SQLAlchemy
Elasticsearch (for search functionality)

Bootstrap (frontend styling)

Docker
